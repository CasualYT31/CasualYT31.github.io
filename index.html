How to Use ▼

Generator Mode

  1. Go through the form and select all of the things you want to start the game with.
  2. As you update the form, you will notice that the password at the bottom of the page will update automatically.
  3. If you are playing on an emulator that lets you edit the game's memory, you can directly insert the copied password into memory instead of typing it out:
    1. First, copy the password as hex.
    2. Then, navigate to the Resume Game menu.
    3. You will need to type in any character. The game won't let you immediately submit a password if your cursor is at the first character.
    4. Navigate to 0x03002d90 in memory (in RALibRetro, this will be 0x002d90).
    5. This points to a small byte array the game uses to keep track of the password the user inputs via the in-game keyboard.
    6. Paste the copied password bytes into the aforementioned address.
    7. The password will not show up in the textbox, but you should be able to submit it anyway.
  4. If you run into a bug with this website, please report it here.

Validator Mode

  1. Type in a password in the textbox at the bottom of the page.
  2. If the password is valid, the read-only form will update with the information encoded within the password, telling you how the game will resume if the password is submitted.
  3. If the password is invalid, the form will disappear until you input a valid password, or switch back to the Generator mode.
  4. This website can also distinguish between a regular password and one of the hard-coded passwords within the ROM. If one is given, the form will be replaced with a brief description of the hard-coded password's effect.
  5. Whitespace will be ignored when typing out a password, but you must remember to use upper case!
