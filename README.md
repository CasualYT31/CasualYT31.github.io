# Butt-Ugly Martians: B.K.M. Battles Password Generator and Validator

[Use the generator now!](https://casualyt31.github.io/)

You can use this website to generate valid passwords for the GBA game. Once submitted, passwords can be used to recall the following game state:

1. The world and level to spawn the player in.
2. The stats of each character:

   a. Their approximate HP.

   b. The number of attack and defense upgrades they have.

   c. The number of attack and defense stars they have.

4. The number of repair kits collected.
5. The difficulty (one of easy, medium or hard).
6. The character that was last selected.
7. The secret gems collected so far.

This means that if you want to unlock features within the Zoom or Doom Gamecube game using the "intended" Gamecube Link method, you can generate a password that unlocks all of the gems for you without needing to find them in game.

This website also provides a validator mode that lets you input any combination of characters to see if it's a valid password in the game. If it is, the website will tell you what information is encoded within it. It will also identify the special hard-coded passwords in the ROM if you provide them, and it will briefly describe what they do. I was able to find some extra passwords in the ROM that haven't been reported elsewhere on the Internet! You can find a complete list of them [here](https://github.com/CasualYT31/CasualYT31.github.io/blob/main/src/main.js#L325-L347).
